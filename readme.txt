Proyecto: portfolio web con HTML, CSS y JS 
Este proyecto se basa en el material de estudio de Arg. Programa 4.0 
También en los trabajos de: 
https://www.freecodecamp.org/news/author/sampurna/ y de: 
https://www.freecodecamp.org/news/author/kolade/
Este portfolio funciona con éxito en: Google Chrome, Firefox y Brave.
En Edge funciona con un defecto: no se sombrea con gris la barra de navegación cuando la página se desplaza 25 pixels en dirección vertical desde el tope.
